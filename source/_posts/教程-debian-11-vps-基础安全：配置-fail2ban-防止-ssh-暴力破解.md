VPS 裸奔在公网上，每天都会有无数个脚本尝试暴力破解你的 SSH 密码。如果你已经修改了 SSH 默认端口（比如改为 \*\*56789\*\*），虽然能躲过大部分扫描，但加上 \*\*Fail2Ban\*\* 才是双重保险。



Fail2Ban 的原理很简单：监控日志 -> 发现有人多次输错密码 -> 自动推算 IP 关进小黑屋。



\### 环境说明



\* 系统：Debian 11

\* SSH 端口：56789 (根据实际情况修改)



\---



\### 第一步：更新源并安装 Fail2Ban



首先更新系统软件包列表，然后直接安装：



\`\``bash

apt update && apt install fail2ban -y
